# Panduan Pemecahan Masalah

## Masalah Umum dan Solusi

### 1. Masalah Registrasi/Login

- **Masalah**: Tidak dapat mendaftar atau masuk.
  - **Solusi**: Pastikan bahwa semua kolom yang diperlukan diisi dengan benar. Periksa email Anda untuk tautan konfirmasi jika Anda baru saja mendaftar.

### 2. Masalah Pembuatan Pemesanan

- **Masalah**: Formulir pemesanan tidak terkirim.
  - **Solusi**: Verifikasi bahwa semua kolom wajib diisi lengkap. Pastikan bahwa koneksi internet Anda stabil.

### 3. Kegagalan Unggah Tanda Tangan

- **Masalah**: Tidak dapat mengunggah tanda tangan.
  - **Solusi**: Periksa format dan ukuran file. Pastikan bahwa Anda menggunakan format gambar yang didukung (misalnya, PNG, JPG) dan ukuran file tidak melebihi batas.

### 4. Melihat Pemesanan

- **Masalah**: Pemesanan tidak ditampilkan.
  - **Solusi**: Refresh halaman atau keluar dan masuk kembali. Jika masalah berlanjut, periksa koneksi internet Anda.

### 5. Masalah Notifikasi

- **Masalah**: Tidak menerima notifikasi.
  - **Solusi**: Periksa folder spam email Anda. Pastikan bahwa pengaturan notifikasi Anda diaktifkan dalam profil akun Anda.

### 6. Akses Dashboard Admin

- **Masalah**: Tidak dapat mengakses dashboard admin.
  - **Solusi**: Konfirmasi bahwa Anda memiliki kredensial admin yang benar. Jika Anda lupa kata sandi, gunakan opsi pemulihan kata sandi.

### 7. Masalah Manajemen Layanan

- **Masalah**: Tidak dapat menambah atau mengedit layanan.
  - **Solusi**: Pastikan bahwa Anda memiliki izin yang diperlukan. Periksa kesalahan validasi pada formulir.

### 8. Kegagalan Pembuatan Laporan

- **Masalah**: Laporan tidak dibuat.
  - **Solusi**: Pastikan bahwa semua filter yang diperlukan diatur dengan benar. Jika masalah berlanjut, coba hapus cache browser Anda.

### 9. Kesalahan Aplikasi Umum

- **Masalah**: Aplikasi crash atau menampilkan pesan kesalahan.
  - **Solusi**: Refresh halaman. Jika masalah berlanjut, hubungi dukungan dengan detail pesan kesalahan.

## Hubungi Dukungan

Jika Anda mengalami masalah yang tidak tercakup dalam panduan ini, silakan hubungi tim dukungan kami di [support@example.com](mailto:support@example.com) untuk bantuan lebih lanjut.
