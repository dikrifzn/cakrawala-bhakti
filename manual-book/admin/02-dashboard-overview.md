# Gambaran Dashboard Admin

Dashboard Admin berfungsi sebagai pusat untuk mengelola platform penyelenggara acara. Dashboard ini memberikan admin gambaran komprehensif tentang metrik utama, notifikasi, dan akses cepat ke berbagai fitur manajemen. Berikut adalah garis besar komponen utama dan fungsi yang tersedia di dashboard.

## Fitur Utama

### 1. Widget Dashboard

Dashboard mencakup beberapa widget yang menampilkan informasi penting secara sekilas, seperti:

- **Total Pemesanan**: Ringkasan jumlah total pemesanan yang dilakukan dalam jangka waktu tertentu.
- **Persetujuan Tertunda**: Jumlah pemesanan yang menunggu persetujuan, memungkinkan admin untuk memprioritaskan tindakan mereka.
- **Aktivitas Terkini**: Log aktivitas terkini yang dilakukan oleh pengguna, termasuk pemesanan, pembatalan, dan pembaruan.

### 2. Menu Navigasi

Menu navigasi di sisi kiri memberikan akses mudah ke berbagai bagian panel admin, termasuk:

- **Manajemen Pemesanan**: Lihat, setujui, atau tolak pemesanan.
- **Manajemen Layanan**: Tambahkan, edit, atau hapus layanan yang ditawarkan di platform.
- **Manajemen Artikel**: Buat dan kelola artikel untuk blog platform.
- **Manajemen Proyek**: Awasi proyek yang sedang berlangsung dan statusnya.
- **Pengaturan Situs**: Konfigurasi pengaturan dan preferensi di seluruh situs.
- **Manajemen Pengguna**: Kelola akun dan peran pengguna.

### 3. Notifikasi

Admin menerima notifikasi real-time untuk acara penting, seperti:

- Pemesanan baru yang memerlukan persetujuan.
- Perubahan yang dilakukan pada pemesanan yang ada.
- Peringatan untuk pembaruan sistem atau pemeliharaan.

### 4. Tindakan Cepat

Dashboard menyediakan tombol tindakan cepat untuk tugas umum, seperti:

- Menambahkan pemesanan baru.
- Membuat layanan baru.
- Membuat laporan.

### 5. Metrik Kinerja

Admin dapat melihat metrik kinerja yang membantu menilai kesuksesan platform, termasuk:

- Tren pemesanan dari waktu ke waktu.
- Pendapatan yang dihasilkan dari pemesanan.
- Statistik keterlibatan pengguna.

## Kesimpulan

Dashboard Admin dirancang untuk merampingkan proses manajemen, memberikan admin alat dan informasi yang mereka butuhkan untuk mengawasi platform penyelenggara acara secara efektif. Dengan memanfaatkan fitur yang tersedia di dashboard, admin dapat memastikan operasi yang lancar dan efisien.
