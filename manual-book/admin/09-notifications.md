# Sistem Notifikasi untuk Admin

## Gambaran Umum

Sistem notifikasi adalah fitur penting dari panel admin yang memungkinkan administrator untuk tetap terinformasi tentang berbagai acara dan pembaruan dalam platform. Dokumen ini menguraikan cara kerja sistem notifikasi, jenis notifikasi yang tersedia, dan cara mengelolanya.

## Jenis Notifikasi

1. **Notifikasi Pemesanan**

   - Admin menerima notifikasi ketika pemesanan baru dibuat, diperbarui, atau dibatalkan. Ini memastikan bahwa mereka selalu mengetahui status terkini dari pemesanan.

2. **Notifikasi Layanan**

   - Notifikasi terkait penambahan, modifikasi, atau penghapusan layanan. Admin diperingatkan tentang perubahan apa pun yang dapat mempengaruhi penawaran pelanggan.

3. **Notifikasi Pengguna**

   - Peringatan mengenai registrasi pengguna, pembaruan, atau masalah apa pun yang terkait dengan akun pengguna. Ini membantu dalam menjaga manajemen pengguna secara efektif.

4. **Notifikasi Sistem**
   - Notifikasi umum tentang pembaruan sistem, jadwal pemeliharaan, atau masalah kritis yang memerlukan perhatian admin.

## Mengelola Notifikasi

### Melihat Notifikasi

- Admin dapat melihat notifikasi dari dashboard. Panel notifikasi khusus menampilkan semua notifikasi terkini, dikategorikan berdasarkan jenis.
- Setiap notifikasi mencakup stempel waktu dan deskripsi singkat tentang acara.

### Menandai Notifikasi

- Notifikasi dapat ditandai sebagai sudah dibaca atau belum dibaca. Admin dapat mengklik pada notifikasi untuk melihat detail lebih lanjut dan menandainya sesuai.

### Pengaturan Notifikasi

- Admin dapat menyesuaikan preferensi notifikasi mereka di bagian pengaturan. Opsi meliputi:
  - Mengaktifkan atau menonaktifkan jenis notifikasi tertentu.
  - Memilih cara notifikasi dikirimkan (misalnya, email, dalam aplikasi).

### Menghapus Notifikasi

- Admin memiliki opsi untuk menghapus notifikasi yang tidak lagi relevan. Ini membantu menjaga panel notifikasi tetap terorganisir.

## Kesimpulan

Sistem notifikasi dirancang untuk meningkatkan kemampuan admin dalam mengelola platform secara efektif. Dengan tetap terinformasi tentang acara dan pembaruan utama, admin dapat merespons dengan cepat dan menjaga operasi platform penyelenggara acara manajemen berjalan lancar.
