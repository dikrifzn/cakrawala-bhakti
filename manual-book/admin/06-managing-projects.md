# Mengelola Proyek

## Pendahuluan

Bagian ini memberikan gambaran tentang cara admin dapat mengelola proyek secara efektif dalam platform Cakrawala Bhakti. Proyek sangat penting untuk mengorganisir acara dan memastikan bahwa semua aspek proses perencanaan acara ditangani secara efisien.

## Mengakses Bagian Manajemen Proyek

Untuk mengelola proyek, navigasi ke dashboard admin dan pilih opsi "Proyek" dari menu. Ini akan membawa Anda ke antarmuka manajemen proyek.

## Membuat Proyek Baru

1. Klik pada tombol "Tambah Proyek Baru".
2. Isi kolom yang diperlukan, termasuk:
   - Nama Proyek
   - Deskripsi
   - Tanggal Mulai
   - Tanggal Selesai
   - Anggota Tim yang Ditugaskan
3. Setelah memasukkan detail, klik "Simpan" untuk membuat proyek.

## Melihat Proyek yang Ada

Dalam antarmuka manajemen proyek, Anda akan melihat daftar semua proyek yang ada. Anda dapat:

- Mencari proyek tertentu menggunakan bilah pencarian.
- Mengurutkan proyek berdasarkan nama, tanggal, atau status.
- Klik pada proyek untuk melihat detailnya.

## Mengedit Proyek

Untuk mengedit proyek yang ada:

1. Pilih proyek dari daftar.
2. Klik pada tombol "Edit".
3. Perbarui kolom yang diperlukan.
4. Klik "Simpan" untuk menerapkan perubahan.

## Menghapus Proyek

Untuk menghapus proyek:

1. Pilih proyek yang ingin Anda hapus.
2. Klik pada tombol "Hapus".
3. Konfirmasi penghapusan dalam prompt yang muncul.

## Pelacakan Status Proyek

Setiap proyek dapat diperbarui statusnya untuk mencerminkan keadaan saat ini. Status yang tersedia meliputi:

- Direncanakan
- Sedang Berlangsung
- Selesai
- Ditahan
- Dibatalkan

Untuk memperbarui status:

1. Pilih proyek.
2. Pilih status baru dari menu dropdown.
3. Klik "Perbarui" untuk menyimpan perubahan.

## Kesimpulan

Mengelola proyek secara efektif sangat penting untuk kesuksesan acara yang diselenggarakan melalui platform Cakrawala Bhakti. Dengan mengikuti langkah-langkah yang diuraikan dalam panduan ini, admin dapat memastikan bahwa semua proyek terorganisir dengan baik dan dilacak sepanjang siklus hidupnya.
