# Instruksi Unggah Tanda Tangan

Dokumen ini memberikan panduan langkah demi langkah bagi pelanggan tentang cara mengunggah tanda tangan mereka dalam platform Cakrawala Bhakti.

## Langkah 1: Akses Bagian Unggah Tanda Tangan

Setelah pemesanan Anda dibuat dan disetujui, Anda akan menerima notifikasi yang meminta Anda untuk mengunggah tanda tangan Anda. Untuk memulai, navigasi ke halaman detail pemesanan.

1. Masuk ke akun Anda di platform Cakrawala Bhakti.
2. Buka bagian "Pemesanan Saya" dari profil Anda.
3. Pilih pemesanan yang memerlukan unggahan tanda tangan.

## Langkah 2: Siapkan Tanda Tangan Anda

Sebelum mengunggah, pastikan bahwa tanda tangan Anda memenuhi persyaratan berikut:

- Tanda tangan harus jelas dan terbaca.
- Harus dalam format file yang didukung (misalnya, PNG, JPEG).
- Ukuran file tidak boleh melebihi 2MB.

## Langkah 3: Unggah Tanda Tangan Anda

1. Di halaman detail pemesanan, temukan tombol "Unggah Tanda Tangan".
2. Klik tombol untuk membuka dialog pemilihan file.
3. Pilih file tanda tangan Anda dari perangkat Anda.
4. Setelah dipilih, klik tombol "Unggah".

## Langkah 4: Konfirmasi Unggahan Anda

Setelah mengunggah, Anda akan melihat pratinjau tanda tangan Anda. Tinjau untuk memastikan itu benar.

- Jika tanda tangan benar, klik tombol "Konfirmasi" untuk menyelesaikan unggahan.
- Jika Anda perlu mengubah tanda tangan, klik tombol "Ubah" untuk memilih file yang berbeda.

## Langkah 5: Penyelesaian

Setelah dikonfirmasi, tanda tangan Anda akan disimpan, dan Anda akan menerima notifikasi yang mengkonfirmasi unggahan yang berhasil. Anda sekarang dapat melanjutkan dengan langkah berikutnya dalam proses pemesanan Anda.

Untuk masalah apa pun selama proses unggahan, silakan lihat bagian pemecahan masalah dalam panduan bersama atau hubungi dukungan pelanggan untuk bantuan.
